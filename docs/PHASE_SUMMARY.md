# ملخص تحديثات تطبيق "حراج سهم" (النسخة الأخيرة)

## المرحلة الأخيرة: الجاهزية للنشر والاشتراكات

1. **شفافية الاشتراكات (متطلبات آبل):** تحديث صفحات الاشتراك لتوضيح الأسعار والمدة بشكل صريح، مع إضافة روابط مباشرة لسياسة الخصوصية وشروط الاستخدام لضمان قبول التطبيق في متجر آبل.
2. **شاشة قسم الذبائح:** إضافة شاشة مخصصة ونظام متكامل لقسم "الذبائح" لتسهيل الوصول إليها وتصفحها بشكل مستقل.
3. **الخرائط وتحديد المواقع:** دمج نظام الخرائط وإضافة خاصية اختيار المدينة والدولة لضمان دقة البحث وتسهيل العثور على الإعلانات القريبة من المستخدم.
4. **نظام تبديل الاشتراكات:** تطوير شامل لنظام تغيير وترقية الباقات، مما يتيح للمستخدمين الانتقال بين الاشتراكات (البرونزية، الفضية، والذبائح) بسهولة ومرونة عالية.
5. **استقرار عمليات الدفع:** إصلاح المشكلات التقنية المتعلقة بـ Apple Pay وبوابات الدفع لضمان إتمام عمليات الشراء والاشتراك دون أي أخطاء.
